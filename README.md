# 📱 SMS Guardian AI

SMS Guardian AI is an intelligent spam detection system that classifies SMS messages as **Spam** or **Not Spam (Ham)** using Machine Learning and Natural Language Processing.

## 🚀 Features
- Detects spam messages with high accuracy
- Uses TF-IDF for feature extraction
- Supports ML models like Naive Bayes / Logistic Regression
- Interactive web interface using Streamlit
- Stores message history using SQL database

## 🛠️ Technologies Used
- Python
- Machine Learning (Scikit-learn)
- NLP (NLTK)
- Streamlit (UI)
- HTML/CSS
- SQL Database

## 📊 Dataset
- SMS Spam Collection Dataset from Kaggle

## ⚙️ How It Works
1. User inputs SMS text
2. Text preprocessing (cleaning, tokenization)
3. Feature extraction using TF-IDF
4. Model predicts Spam or Ham
5. Result displayed on UI
